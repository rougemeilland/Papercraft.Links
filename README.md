## 【付録】

### 両面印刷のテスト

PDFの両面印刷のずれの有無を確認します。

以下のPDFファイルをダウンロードしてご使用のプリンタで印刷してください。

- [ダウンロード (A4 カラー 2ページ)](https://drive.google.com/file/d/1S2fynfQ8sFoITRBwmPuqkFic-Ln867OD/view?usp=sharing)

印刷には以下の指定をしてください。

| 項目  | 設定値 |
|--|--|
| 原稿の向き | 縦 |
| 原稿サイズ | A4 |
| 印刷種類 | 両面 |
| 開き/綴じ方向 | 長辺綴じ(左綴じ) 
| カラー/モノクロ | カラー |

印刷が終わったら、表面と裏面がずれていないかどうかを確認してください。
